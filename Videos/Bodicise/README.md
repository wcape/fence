# Bodicise - AI-Powered Fitness Platform

A modern, responsive fitness platform built with HTML, CSS, and JavaScript, featuring AI-powered workout generation, real-time form analysis, and comprehensive progress tracking.

## ğŸ—ï¸ Architecture Overview

This project follows a **Microsoft MVC framework structure** with separate controller, view, and model files as HTML files with embedded scripts/functions, all served from a main Google Apps Script backend.

### Frontend (GitHub Pages)
- **Static HTML/CSS/JS** hosted on GitHub Pages
- **MVC Structure**: Separate controller, view, and model files
- **Responsive Design**: Mobile-first approach with modern UI/UX
- **Progressive Web App (PWA)** capabilities

### Backend (Google Cloud)
- **Google Apps Script** for main backend logic
- **Stripe Integration** for secure payments
- **AI Model Hosting** on Vertex AI
- **Database**: Cloud Firestore for real-time data
- **Authentication**: Firebase Auth

## ğŸ”§ Setup Instructions

### 1. Frontend Setup (GitHub Pages)

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/bodicise.git
   cd bodicise
   ```

2. **Enable GitHub Pages**:
   - Go to repository Settings
   - Scroll to Pages section
   - Select source as "Deploy from a branch"
   - Choose "main" branch and "/ (root)" folder

3. **Custom Domain** (Optional):
   - Add CNAME file with your domain
   - Configure DNS settings

### 2. Backend Setup (Google Cloud)

1. **Create Google Cloud Project**:
   - Go to [Google Cloud Console](https://console.cloud.google.com)
   - Create new project
   - Enable required APIs

2. **Setup Google Apps Script**:
   - Create new Apps Script project
   - Copy backend code from `backend/` folder
   - Deploy as web app

3. **Configure Services**:
   - Setup Cloud Firestore database
   - Configure Firebase Authentication
   - Setup Vertex AI for AI models
   - Configure Stripe webhooks

### 3. Environment Configuration

Update the following in your frontend code:

```javascript
// In assets/js/app.js
App.config = {
    apiBaseUrl: 'https://your-google-cloud-function-url.com/api',
    stripePublishableKey: 'pk_live_your_stripe_publishable_key',
    environment: 'production'
};
```

## ğŸ“ API Documentation

### Authentication Endpoints
- `POST /auth/login` - User login
- `POST /auth/register` - User registration
- `POST /auth/logout` - User logout
- `GET /auth/verify` - Verify token

### User Endpoints
- `GET /user/profile` - Get user profile
- `PUT /user/profile` - Update user profile
- `GET /user/stats` - Get user statistics
- `GET /user/progress` - Get progress data

### Workout Endpoints
- `GET /workouts` - Get workout library
- `POST /workouts/generate` - Generate AI workout
- `POST /workouts/start` - Start workout
- `POST /workouts/finish` - Finish workout

### Payment Endpoints
- `POST /payment/create-intent` - Create payment intent
- `POST /payment/confirm` - Confirm payment
- `GET /payment/history` - Get payment history
